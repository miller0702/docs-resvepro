# Logos RESVEPRO

Assets oficiales de marca para la plataforma **RESVEPRO** (*Restaurando Verdades Proféticas*).

| Archivo | Uso |
|---------|-----|
| `LogoFondosBlancosLetra.png` | Logo completo con nombre y tagline (login, splash, marketing) |
| `LogoFondosBlancos.png` | Isotipo (icono de app, favicon, sidebar) |

Copias desplegadas en:

- `egw-admin/public/logo-full.png` y `logo-icon.png`
- `egw-mobile/assets/logo-full.png`, `logo-icon.png`, `icon.png`, `adaptive-icon.png`, `splash-icon.png`

Al actualizar estos archivos, vuelve a copiarlos a admin y mobile.
